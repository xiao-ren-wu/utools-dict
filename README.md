# 数据字典管理器 (Dict Manager)

一个简单高效的数据字典管理工具，让数据管理变得轻而易举！

## ✨ 特色功能

### 1. 便捷的数据录入 (dictinput)
- 支持 JSON 数组格式批量录入
- 支持数据分类管理
- 录入后即可快速检索
- **支持超链接格式**：`[显示名称|链接地址]`

示例数据格式：
```json
[
  {"name": "张三", "age": 25, "job": "工程师", "website": "[个人主页|https://example.com]"},
  {"name": "李四", "age": 28, "job": "设计师", "email": "[联系邮箱|mailto:lisi@example.com]"}
]
```

**超链接功能说明：**
- 格式：`[显示名称|链接地址]`
- 搜索时只搜索显示名称部分
- 显示时只显示超链接名称，带有蓝色下划线
- 点击超链接会在新窗口打开对应地址
- 支持各种链接类型：http/https、mailto、tel等

### 2. 智能搜索 (dict)
- 支持关键字 + 模糊搜索
- 实时搜索结果展示
- 支持多字段匹配
- 搜索结果表格化展示
- **超链接搜索优化**：搜索时只匹配显示名称部分

使用方式：`dict 关键字:搜索内容`  
例如：`dict 员工:工程师` 或 `dict 员工:GitHub`（搜索超链接显示名称）
![dict](/docs/dict.png)
### 3. 数据管理 (dictlist)
- 分类展示所有数据
- 支持按创建时间排序
- 支持批量删除记录
- 支持分类数据清理
- 横向滚动查看完整内容
- **超链接渲染**：自动识别并渲染超链接格式

   - 客户资料整理（包含联系方式）

2. **快速检索查询**

   - 分类数据查看

   - **超链接名称搜索**

3. **临时数据管理**
   - 会议记录整理
   - 项目信息收集
   - 临时数据存储
   - **资源链接管理**

## 🚀 使用方法

1. 数据录入：
   - 输入 `dictinput`
   - 粘贴 JSON 数组数据
   - 输入分类关键字
   - 点击提交保存
   - **超链接格式**：`[显示名称|链接地址]`

2. 数据搜索：
   - 输入 `dict`
   - 输入 `关键字:搜索内容`
   - 实时展示匹配结果
   - **搜索超链接时只需输入显示名称**

3. 数据管理：
   - 输入 `dictlist`
   - 查看所有分类数据
   - 可进行批量操作
   - **点击超链接直接打开对应地址**
